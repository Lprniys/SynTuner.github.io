Implementation of Oracle Turing Machines 11/6/2014

This is an extension of the implementation of Turing machines written in OCaml. Oracle machine is a central concept in modern computability theory. This is a program which could actually run a well-defined oracle Turing machine program. In this sense, it is also a compiler for oracle Turing machines. For details and how to use see the README here.
